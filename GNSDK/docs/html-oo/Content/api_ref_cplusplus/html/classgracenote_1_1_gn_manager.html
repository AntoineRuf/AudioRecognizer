Detailed Description

Provides core functionality necessary to all Gracenote objects. This class must be instantiated prior to any other Gracenote objects.

Creates a new Serialized User and also increments the user's Client ID user count with Gracenote Service. Use this constructor to create a new user; when successful, this call registers a new user for a specified client in Gracenote Service. Once the new user is registered and the user count incremented in Gracenote Service, the count cannot be reduced, nor can the same user be again retrieved. Newly registered user handles must be serialized and stored locally for that user to be used again for future queries; failing to do this quickly depletes the client's allotted user quota within the Gracenote Service.

Parameters
registerMode[in] register as online or local only user
clientId[in] client ID that initiates requests with this handle; value provided by Gracenote
clientTag[in] client ID tag value that matches client ID; value provided by Gracenote
applicationVersion[in] client application version; numeric value provided by application, and this value is required
