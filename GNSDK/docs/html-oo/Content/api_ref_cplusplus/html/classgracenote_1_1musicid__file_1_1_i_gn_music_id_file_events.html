gnsdk_uint32_t  currentFile,
gnsdk_uint32_t  totalFiles,
IGnCancellable &  canceller 
)
pure virtual

Callback function declaration for a fingerprint generation request.

Parameters
fileInfo[in] FileInfo object that the callback operates on
currentFile[in] Current number of the file being processed
totalFiles[in] Total number of files to be processed
canceller[in] Object that can be used to cancel processing if desired

Remarks:

The application can implement this callback to use the fingerprint_begin(), fingerprint_write() and fingerprint_end() APIs with the given FileInfo object to generate a fingerprint from raw audio. If your application already has created the fingerprint for the respective file, the application should implement this callback to use the GnMusicIdFileInfo.set*() API to set any metadata values in the FileInfo object. This callback is called only if no fingerprint has already been set for the passed FileInfo object.
