Detailed Description

Experimental: Provides access to information regarding the underlying GNSDK playlist library. For working with user collections to create playlists see GnPlaylistCollection.

Retrieves the Playlist SDK's build date string.

Returns

static GNWRAPPER_ANNOTATE gnsdk_cstr_t gracenote::playlist::GnPlaylist::Version ( )
static

Retrieves the Playlist SDK version string.

Returns
version string if successful
GNSDK_NULL if not successful

Remarks:

This API can be called at any time, after getting instance of GnManager successfully. The returned string is a constant. Do not attempt to modify or delete. Example version string: 1.2.3.123 (Major.Minor.Improvement.Build) Major: New functionality Minor: New or changed features Improvement: Improvements and fixes Build: Internal build number
