gnsdk_cstr_t gracenote::metadata::GnVideoWork::TuiMatchProduct ( ) const
inline

Second TUI, if it exists. This TUI is used for matching partial Products objects to full Works objects. Use this value to ensure correct Tui value matching for cases when a video Product GDO contains multiple partial Work GDOs. Each partial Work GDO corresponds to a full Works object, and each full Works object contains the GNSDK_GDO_VALUE_TUI value, incremented by one digit to maintain data integrity with Gracenote Service. The GNSDK_GDO_VALUE_TUI_MATCH_PRODUCT maps the partial Works object Tui value to the full Works object.
