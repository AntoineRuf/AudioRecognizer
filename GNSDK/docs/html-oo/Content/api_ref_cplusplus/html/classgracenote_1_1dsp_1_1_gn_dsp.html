Retrieves the GnDsp SDK's build date string.

Returns
gnsdk_cstr_t Build date string of the format: YYYY-MM-DD hh:mm UTC

Remarks:

This API can be called at any time, after getting instance of GnManager successfully. The returned string is a constant. Do not attempt to modify or delete. Example build date string: 2008-02-12 00:41 UTC

static gnsdk_cstr_t gracenote::dsp::GnDsp::Version ( )
static

Retrieves GnDsp SDK version string. This API can be called at any time, after getting instance of GnManager successfully. The returned string is a constant. Do not attempt to modify or delete. Example: 1.2.3.123 (Major.Minor.Improvement.Build) Major: New functionality Minor: New or changed features Improvement: Improvements and fixes Build: Internal build number
