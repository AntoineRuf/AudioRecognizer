static gnsdk_cstr_t gracenote::link::GnLink::Version ( )
static

Retrieves the Link library version string. This API can be called at any time, after getting GnManager instance successfully. The returned string is a constant. Do not attempt to modify or delete.

Example: 1.2.3.123 (Major.Minor.Improvement.Build)
Major: New functionality
Minor: New or changed features
Improvement: Improvements and fixes
Build: Internal build number
