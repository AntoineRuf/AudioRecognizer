Detailed Description

Provides access to content assets that can be retrieved from Urls. Content is only available is content is enabled from the original query's lookup data. Use the query object's options to enable content. The type of content that is available depends on the parent object. For example you can access album cover art from a GnAlbum object, but not an artist image. To retrieve an artist image navigate to a GnContributor object and invoke it's GnContent instance.
