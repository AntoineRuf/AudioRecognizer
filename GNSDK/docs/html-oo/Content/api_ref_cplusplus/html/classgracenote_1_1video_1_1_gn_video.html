static gnsdk_cstr_t gracenote::video::GnVideo::BuildDate ( )
static

Retrieves the VideoID and Video Explore library's build date as a string. This API can be called at any time, even before initialization and after shutdown. The returned string is a constant. Do not attempt to modify or delete.

Returns
String with format: YYYY-MM-DD hh:mm UTC

Example:"2008-02-12 00:41 UTC"
