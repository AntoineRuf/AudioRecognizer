void gracenote::link::GnLinkOptions::Clear ( )
throw (GnError
)

Clears all options currently set for a given Link query.

Remarks:

As Link query handles can be used to retrieve multiple enhanced data items, it may be appropriate to specify different options between data retrievals. You can use this function to clear all options before setting new ones.

void gracenote::link::GnLinkOptions::NetworkInterface ( gnsdk_cstr_t  ipAddress)
throw (GnError
)

This option allows setting of a specific network interface to be used with connections made by this object. Choosing which interface to use can be beneficial for systems with multiple network interfaces. Without setting this option, connections will be made of the default network interface as decided by the operating system.

Parameters
ipAddress[in] local IP address for the desired network interface
