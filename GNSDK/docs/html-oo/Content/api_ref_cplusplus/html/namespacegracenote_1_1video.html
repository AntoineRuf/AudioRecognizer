enum gracenote::video::GnVideoSearchType

The type of text search that is used to find results.

Enumerator
kSearchTypeUnknown 

Unknown search type; the default state.

kSearchTypeAnchored 

Anchored text search, used for product title and suggestion searches. Retrieves results that begin with the same characters as exactly specified; for example, entering dar, dark, dark k, or dark kni retrieves the title Dark Knight, but entering knight will not retrieve Dark Knight. Note that this search type recognizes both partial and full words.

kSearchTypeDefault 

Normal keyword filter search for contributor, product, and work title searches; for example, a search using a keyword of dark, knight, or dark knight retrieves the title Dark Knight. Note that this search type recognizes only full words, not partial words; this means that entering only dar for dark is not recognized.

enum gracenote::video::GnVideoTOCFlag

The TOC Flag type.

Enumerator
kTOCFlagDefault 

Generally recommended flag to use when setting a video TOC.

Remarks:

Use this flag for the TOC flag parameter that is set with the FindProducts() API, for all general cases; this includes when using the Gracenote Video Thin Client component to produce TOC strings. For cases when other VideoID TOC flags seem necessary, contact Gracenote for guidance on setting the correct flag.

kTOCFlagPal 

Flag to indicate a given simple video TOC is from a PAL (Phase Alternating Line) disc.

Remarks:

Use this flag only when directed to by Gracenote. Only special video TOCs that are provided by Gracenote and external to the GNSDK may require this flag.

kTOCFlagAngles 

Flag to indicate a given simple video TOC contains angle data.

Remarks:

Use this flag only if Gracenote directs you to. Only special video TOCs that Gracenote provides and that are external to the GNSDK may require this flag.
