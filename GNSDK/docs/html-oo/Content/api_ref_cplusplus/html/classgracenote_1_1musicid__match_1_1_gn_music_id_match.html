Detailed Description

MusicID Match uses a combination of waveform fingerprinting technologies to match tracks within an end-user's collection to tracks within a cloud music provider's catalog, enabling instant playback from all devices without requiring upload.

The GnMusicId and GnMusicIdFile classes provide methods for generating fingerprints from audio files and streaming music.
