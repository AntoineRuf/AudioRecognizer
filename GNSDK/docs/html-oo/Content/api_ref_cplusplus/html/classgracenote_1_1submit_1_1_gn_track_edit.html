gnsdk_cstr_t gracenote::submit::GnTrackEdit::Year ( )

Sets a list-based value by the list item Submit ID for YEAR. If the value does not exist, it is added. If the value does exist, it is changed. If the role_submit_id is 0, the value is deleted.

Parameters
valueset Value corresponding to the specified GnDataObject value key

Remarks:

Use this function to set a list-based Submit ID to YEAR, prior to adding the GnDataObject to a parcel.
