Classical Album Metadata Example

The following is intended to give the reader some idea of the metadata returned in a GDOGracenote Data Object: Containers most-commonly used to store information returned by a query. GDOs can contain metadata values (names, titles, external IDs, and so on) that can be accessed by an application using Value keys. GDOs can also contain references to other GDOs, which can be accessed using Child GDO keys. for a classical album.
